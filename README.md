# 발로란트 방송 오버레이 (VCT 퍼시픽 스타일)

개인 방송인이 OBS에 송출할 수 있는 발로란트 대회용 오버레이입니다.
표시 요소: 팀명·팀로고 · 스코어보드 · 맵 획득(마름모) · 맵/라운드 · 공수(ATK/DEF) ·
선수 명단 · 타이머 · 무기/방어구 · 궁극기 포인트 · 스킬(C/Q/E) · 선수 캠 슬롯.

---

## 두 가지 사용 방법 — 하나만 고르면 됩니다

### ⭐ 방법 A. 무설치 (추천 · 비개발자용)
**아무것도 설치하지 않습니다.** `valorant-overlay-app.html` 파일 하나를 OBS에
"브라우저 소스 → 로컬 파일"로 넣기만 하면 됩니다.
- 따라하기: **`START-HERE-무설치.html`** 를 더블클릭해서 4단계 가이드를 보세요.
- 값 변경은 OBS에서 소스 우클릭 → **"상호작용"** 으로. 단축키(Q/P/S/Space 등)도 작동.
- 컨트롤 패널은 화면 아래(1080 밖)에 숨겨져 방송에는 안 나옵니다.
- OBS 브라우저 소스 크기는 **1920 × 1480** 로 설정 (높이 1080 아님 주의).

### 🛠 방법 B. 설치형 앱(.exe) — 분리된 컨트롤 창 + 투명 오버레이
별도 창/태블릿에서 편하게 조작하고 싶을 때. 완성된 `ValorantOverlay.exe`는
**더블클릭만 하면** 서버가 켜지고 컨트롤 패널이 자동으로 열립니다(받는 사람은 설치 0).
- exe 만드는 법: **`BUILD-EXE-안내.md`** 참고 (GitHub 클라우드 빌드 또는 `build-exe.bat` 더블클릭).
- 만든 뒤: exe 더블클릭 → OBS 브라우저 소스에 `http://localhost:8080/overlay`(1920×1080) → 자동으로 열린 컨트롤 패널에서 조작.
- Node를 직접 다룰 줄 알면 빌드 없이 `node server.js` 로 바로 실행해도 됩니다(아래 "설치형 실행").

> 둘은 동일한 디자인·기능입니다. 차이는 "조작 방식"뿐입니다.
> A는 OBS 상호작용 창에서, B는 완전히 분리된 브라우저 창/폰에서 조작합니다.

## 설치형 실행 (방법 B)

---

## ⚠️ 먼저 알아둘 것 — 데이터는 "수동 입력"입니다

발로란트의 무기·궁극기·체력 같은 **실시간 인게임 데이터를 자동으로 받아오는 공식 API는
라이엇 방송 파트너에게만 제공**됩니다. 개인은 접근할 수 없습니다.

따라서 이 프로그램은 **운영자가 경기를 보면서 컨트롤 패널에서 직접 값을 바꾸는 방식**입니다.
혼자 방송하며 동시에 모든 값을 입력하긴 어려우므로, 보통은 ① 스코어/공수/맵 정도만
간단히 갱신하거나, ② 친구 1명이 옵저버 역할로 패널을 조작하는 식으로 운영합니다.

---

## 설치 & 실행 (3단계)

1. **Node.js 설치** — https://nodejs.org (LTS 버전). 추가 라이브러리 설치 불필요.
2. 이 폴더에서 터미널을 열고:
   ```
   node server.js
   ```
3. 터미널에 표시되는 주소를 사용합니다:
   - **OBS 브라우저 소스**: `http://localhost:8080/overlay`
   - **컨트롤 패널(운영자)**: `http://localhost:8080/control`
   - 폰/다른 PC에서 조작하려면 표시된 `http://192.168.x.x:8080/control` 주소 사용

---

## OBS 설정

1. 소스 → **추가 → 브라우저** 선택.
2. URL에 `http://localhost:8080/overlay` 입력.
3. 너비 **1920**, 높이 **1080** 입력 → 확인.
4. 게임 캡처 소스 **위에** 오버레이가 오도록 순서 정렬.
5. 배경은 자동으로 투명하므로 게임 화면이 비칩니다.

자세한 점검은 `OBS-setup-checklist.md` 참고.

## 사용법

### 혼자 운영할 때 (권장 워크플로)
컨트롤 패널 맨 위 **퀵 컨트롤**만 쓰면 됩니다. 게임/캐스팅 중에도 **키보드 단축키**로 빠르게:

| 키 | 동작 | 키 | 동작 |
|----|------|----|------|
| `Q` | 좌측 팀 라운드 승 (스코어+1, 라운드+1) | `Space` | 타이머 시작/정지 |
| `P` | 우측 팀 라운드 승 | `X` | 스파이크 설치/해제 |
| `S` | 공수 좌우 교체 | `R` | 타이머 1:40 | 
| `B` | 타이머 0:30(구매) | | |

선수 10명 정보를 못 채울 땐 **오버레이 표시 → 선수 레일 OFF** 로 스코어바만 깔끔하게 송출하세요.

### 옵저버가 있을 때
하단 **선수 명단**에서 양 팀 5명씩 무기·HP·방어구·크레딧·궁극기·스킬(C/Q/E)을 실시간 갱신합니다.

### 전체 항목
- **경기 상태**: 맵, 형식(Bo1/Bo3/Bo5), 라운드, 국면, 타이머.
- **팀 A/B**: 팀명, 약칭, 컬러, 로고 URL, 스코어(+/−), **맵 획득(+/−)**.
- **맵 획득**: 팀 로고 아래 마름모 칸으로 표시. Bo3는 2칸, Bo5는 3칸이 생기고, 이긴 만큼 팀 컬러로 채워집니다. 새 맵 시작 시 퀵 컨트롤의 "↻ 새 맵 시작"으로 스코어를 0-0으로 리셋(맵 획득 수는 유지).
- **캠**: ON/OFF, 좌하단·우하단, 라벨. 켜면 빈 프레임이 생기고 OBS에서 실제 캠 소스를 그 위에 올립니다.

## 커스터마이징

- **색·폰트·크기**: `public/overlay.css` 상단의 CSS 변수(`:root`)에서 조정.
- **에이전트/무기/맵 목록**: `public/shared.js`의 `AGENTS`, `WEAPONS`, `MAPS` 배열 수정.
- **팀 로고**: 컨트롤 패널의 "로고" 칸에 이미지 URL을 넣거나, 로컬 파일은 `public/` 폴더에
  넣고 `http://localhost:8080/파일명.png` 형태로 입력.

## 구성 파일

```
valorant-overlay/
├─ server.js               # 무의존성 Node 서버 (정적 제공 + 상태 중계, SSE)
├─ package.json
├─ demo.html               # 단일 파일 미리보기(서버 없이 브라우저로 바로 열기)
├─ README.md
├─ OBS-setup-checklist.md
└─ public/
   ├─ overlay.html         # OBS 브라우저 소스
   ├─ overlay.css          # 디자인 토큰 + 컴포넌트
   ├─ overlay.js           # 렌더링 + 실시간 수신(SSE)
   ├─ control.html         # 운영자 컨트롤 패널
   ├─ control.js           # 패널 로직(상태 전송)
   └─ shared.js            # 기본 상태/목록 데이터
```

## 동작 원리

`control.js`가 변경된 전체 상태를 `POST /update`로 서버에 보내고,
서버는 그 상태를 **SSE(`/events`)** 로 연결된 모든 오버레이에 즉시 전파합니다.
별도 데이터베이스나 외부 서비스 없이 로컬에서 완결됩니다.

> 팁: 미리보기만 빠르게 보고 싶으면 `demo.html`을 브라우저로 그냥 더블클릭하세요.
> (서버 없이 컨트롤이 내장된 데모 버전)
