# 연결 문제 해결 가이드

컨트롤 패널/오버레이 URL이 안 열릴 때, 아래를 위에서부터 확인하세요.
**거의 대부분은 "서버가 안 켜져 있어서"입니다.** URL은 서버가 켜져 있을 때만 열립니다.

---

## 0. 핵심 개념
이 프로그램은 `http://localhost:8080/...` 주소를 **`server.js`가 실행 중일 때만** 제공합니다.
서버를 안 켜고 주소만 입력하면 브라우저가 "연결할 수 없음(ERR_CONNECTION_REFUSED)"을 띄웁니다.
→ **항상 먼저 `node server.js`로 서버를 켜고, 그 창을 닫지 않은 채** 주소를 엽니다.

---

## 1. 증상별 원인 찾기

### A. 브라우저가 "사이트에 연결할 수 없음 / 연결이 거부됨"
서버가 안 켜졌거나 꺼졌습니다.
1. 터미널(명령 프롬프트)을 열었나요? **파일을 더블클릭하면 안 됩니다.**
2. `valorant-overlay` 폴더 안에서 `node server.js`를 실행했나요?
3. 터미널에 `서버 실행 성공!` 배너가 떴나요?
   - 떴다면: 주소를 정확히 `http://localhost:8080/control` 로 입력(오타·앞 공백 주의).
   - 안 떴고 빨간 `✗ [포트 사용 중]` → 2번 항목 참고.
   - 안 떴고 `✗ [실행 위치 오류]` → 폴더가 틀렸습니다. server.js가 있는 폴더에서 실행.
   - `'node'은(는) 내부 또는 외부 명령... / command not found` → Node.js 미설치. 3번 참고.

### B. 페이지는 열리는데 우측 상단이 "● 연결 안 됨"
HTML 파일을 **더블클릭해서 file:// 로 연** 경우입니다.
→ 더블클릭 대신, 서버를 켠 뒤 브라우저 주소창에 `http://localhost:8080/control` 입력.

### C. 폰/다른 PC에서 안 열림
폰에서 `localhost`는 폰 자신을 가리킵니다.
→ 서버 띄운 PC의 주소(터미널에 출력된 `http://192.168.x.x:8080/control`)를 입력하고,
   **둘이 같은 와이파이**에 있어야 합니다. 안 되면 PC 방화벽에서 Node 허용.

---

## 2. 포트 8080이 사용 중일 때
터미널에 `✗ [포트 사용 중]`이 뜨면 다른 포트로 실행하세요.
- **Windows**: `set PORT=8090 && node server.js`
- **Mac/Linux**: `PORT=8090 node server.js`

그러면 주소도 바뀝니다 → `http://localhost:8090/control`, OBS도 `http://localhost:8090/overlay`.

---

## 3. 터미널에서 서버 켜는 법 (처음이라면)

### Windows
1. `valorant-overlay` 폴더를 파일 탐색기로 엽니다.
2. 주소 표시줄(폴더 경로 칸)을 클릭하고 `cmd` 입력 후 Enter → 그 폴더에서 명령 프롬프트가 열립니다.
3. `node -v` 입력 → 버전(예: v20.x)이 나오면 설치 OK. `'node'...명령이 아닙니다`가 나오면 https://nodejs.org 에서 LTS 설치 후 PC 재시작.
4. `node server.js` 입력 → `서버 실행 성공!` 확인.
5. **이 창을 그대로 둔 채** 브라우저에서 주소를 엽니다.

### Mac
1. `valorant-overlay` 폴더를 Finder에서 엽니다.
2. 폴더를 우클릭 → "폴더에서 새로운 터미널 열기"(또는 터미널을 열고 `cd ` 입력 후 폴더를 드래그&드롭).
3. `node -v` 확인 → 없으면 nodejs.org 에서 설치.
4. `node server.js` 실행 → `서버 실행 성공!` 확인.
5. 창을 둔 채 브라우저에서 주소 열기.

---

## 4. 그래도 안 되면 알려줄 정보
다음을 알려주시면 정확히 짚어드립니다.
1. `node -v` 출력값
2. `node server.js` 실행 시 터미널에 나온 **전체 메시지**(성공 배너인지, ✗ 에러인지)
3. 브라우저에 입력한 주소와, 화면에 뜬 **정확한 오류 문구**
4. OS(Windows/Mac), 그리고 같은 PC에서 여는지 폰에서 여는지
